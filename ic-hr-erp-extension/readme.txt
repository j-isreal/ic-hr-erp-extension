Adds applicant tracking and employee portal to WP ERP Free version.

== Description ==

= Features =
* Complete applicant tracking system
* Secure employee portal
* Job application forms with CAPTCHA
* Email notifications in English and Spanish
* Employee role management

= Requirements =
* WP ERP Free plugin
* Simple Cloudflare Turnstile plugin

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ic-hr-erp-extension` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the settings under HR Management â†’ Settings
4. Add [applicant_form] shortcode to a page for applications
5. Add [employee_portal] shortcode to a page for employee access
